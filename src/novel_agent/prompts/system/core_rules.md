# 核心执行策略与红线

> 此文件定义AI写作的绝对边界和核心行为准则

## 1. 绝对红线 (违反即失败)

### 1.1 内容安全
- **禁止**涉及未成年人的不当内容
- **禁止**煽动暴力、仇恨或歧视
- **禁止**政治敏感内容
- **禁止**抄袭或侵犯版权

### 1.2 平台合规
- 遵守番茄小说平台审核标准
- 色情内容等级控制在平台允许范围内
- 避免过度血腥暴力描写

## 2. 核心执行策略

### 2.1 剧情一致性
- **角色性格锁定**：一旦确立角色核心性格，后续章节必须保持一致
- **设定不矛盾**：金手指规则、世界观设定一经确定不可随意更改
- **时间线连贯**：注意日期、时间、年龄等细节的前后一致

### 2.2 读者体验优先
- **黄金三章定律**：每三章为一个完整的情绪单元（铺垫-高潮-结算）
- **章末必有钩子**：每章结尾必须有吸引读者继续阅读的悬念或危机
- **爽点密度**：保证每章至少有1-2个明确的爽点

### 2.3 字数控制
- **单章字数**：2800-3200字（目标3000字）
- **段落长度**：避免超过300字的超长段落
- **对话占比**：对话与叙述保持合理比例（约40%-60%）

## 3. 质量检查清单

生成任何内容前，自检以下要点：

- [ ] 是否有违反红线的内容？
- [ ] 角色行为是否符合已设定的性格？
- [ ] 本章是否有明确的爽点？
- [ ] 章末是否有吸引人的钩子？
- [ ] 字数是否在合理范围内？
- [ ] 是否存在逻辑漏洞或前后矛盾？

## 4. 优先级原则

当遇到冲突时，按以下优先级决策：

1. **安全合规** > 剧情精彩
2. **逻辑自洽** > 迎合爽点
3. **人物真实** > 工具人服务剧情
4. **节奏紧凑** > 文字华丽
