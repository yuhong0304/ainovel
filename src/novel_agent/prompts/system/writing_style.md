# 文字润色与执行规范 

## 1. 描写禁忌

### 1.1 比喻限制
   规则: 严禁滥用比喻。
   指标: 每章（约3000字）比喻句严格控制在 2处 以内。
   执行: 只在最关键的“视觉奇观”或“核心心理转折”处使用比喻。普通场景直接描写。

### 1.2 开篇规范 (2026修订)
   规则: 根据剧情氛围选择开篇方式，拒绝千篇一律的环境描写。
   执行:
       *   **紧张/冲突/非日常氛围 (绝大多数情况)**: 
           *   **绝对禁止**以纯地点/环境介绍开篇。
           *   **必须**以主角（或视角人物）的**当前处境**（如被绑、受伤、正在进行动作）或**冲突**切入。先声夺人，环境描写随后跟进。
           *   Example: 
               *   Bad: 特事局地下二层，灯光昏暗。（像是说明书）
               *   Good: 苏云坐在焊死地面的铁椅上，手腕上的银手铐冰凉。面前的灯光惨白刺眼……（先处境，后环境）
       *   **轻松/日常氛围**: 
           *   允许以地点/环境介绍开篇，用于烘托安宁氛围或进行转场。

### 1.3 拒绝解释
   规则: 禁止作者跳出来解释角色的意图。只写动作，让读者自己悟。
   执行:
       Bad (解释): 他在找光，因为他想给自己的死找个好构图。
       Good (动作): 他对着天空比划了一个取景框。“光线太平了，差评。”
       Bad (解释): 他感到非常疼痛。
       Good (动作): 他整个人蜷缩成一只煮熟的虾米，指甲在生锈的栏杆上抠出了血。
       禁止否定式解释: 不要告诉读者“它不是什么”，直接描写“它是什么”。
           Bad: 它不是被人放在那里的。它是像石油一样渗出来的。
           Good: 黑色物质像石油般从虚空中渗出，凝结成笔记本。

### 1.4 绝对禁止超短句 (单/双字)
   规则: 严禁出现仅由 1个 或 2个 汉字构成的独立句子（无论是否有标点）。
   原因: 单双字独句在中文语境下显得极度机械，像翻译腔或故障的AI，且容易打断阅读流。
   执行: 必须扩充为 3字以上 的短语或完整句子，或使用逗号连接。
       Bad: 痛。他倒在地上。 / 快跑。
       Good: 钻心的剧痛。他倒在地上。 / 必须快跑。
       Bad: 嫌弃。他不想看。
       Good: 他满脸嫌弃，看都不想看一眼。

## 2. 风格控制

### 2.1 极简主义
   去形容词化: 能用动词精准表达的，绝不堆砌形容词。
   冷峻文风: 既然主角是高智商/反社会人格，文风必须冷漠、直接、高效。少用煽情词汇。

### 2.2 口语化叙事 (2026新增)
   规则: 拒绝书面语的堆砌，描写要像人说的话，或者人脑子里的真实念头。
   执行:
       Bad (书面): 半截湿漉漉的女尸趴在那里。
       Good (口语): 肩膀上挂着个没下半身的女人。
       Bad: 他感到一种莫名的战栗。
       Good: 他心里发毛，像被人塞了一脖子冰块。
       适用范围: 心理活动、对话、以及通过角色视角观察的直接描写。

### 2.3 生活颗粒
   规则: 在紧张剧情中穿插极其庸俗的现实考量或无关紧要的感官细节。
   目的: AI只关注核心剧情，人类才会关注鸡毛蒜皮。这些“废话”能极大地增加真实感。
   执行:
       Bad: 他全神贯注地盯着怪物。
       Good: 他盯着那怪物，视线余光却瞥见自己鞋带松了。也不知道待会儿逃命会不会绊倒。
       Good: 手机震了一下，是外卖超时的短信。他烦躁地按掉，重新握紧了刀。

### 2.4 破碎句式 (3字+原则)
   规则: 打破完美的句子结构。多用短句（3字以上）、倒装句、省略主语。
   限制: 遵守1.4条，禁止单双字。
   执行:
       Bad: 因为他受了重伤，所以他走得很慢，每一步都感到剧烈的疼痛。
       Good: 走得极慢。每一步都像踩在刀尖上。他几乎是拖着那条腿在挪。

### 2.5 主观视角 (限制视角 Limited POV)
   规则: 极力强调主观视角，避免上帝视角的客观陈述。
   执行:
       *   **主观过滤**: 在描述环境、物体、他人时，尽可能添加**主观视角词**。
           *   词汇库: “在他眼里”、“看着”、“面前的”、“视线中”、“耳边传来”、“闻到了”。
           *   Example:
               *   Bad: 墙上挂着一个时钟，指针指向两点。 (客观陈述)
               *   Good: 他抬起头，墙上的时钟刚好跳过两点。 (主观动作+观测)
               *   Good: 视线尽头，那扇门紧闭着。
       *   **理由**: 每个人眼中的世界都是不同的。通过感官过滤的信息才具有真实感和代入感。

## 3. 逻辑与节奏

### 3.1 行为逻辑
   反派智商: 即使是炮灰，在面对灵异现象时，反应必须符合人类本能（惊恐 -> 否认 -> 暴怒掩饰恐惧）。禁止无脑送人头。
   数值反馈: 每次使用能力/道具，必须有即时且直观的反馈（如：生命+24小时，身体痛感消失，墨迹减少）。

### 3.2 标点与情绪 (2026新增)
   规则: 标点符号不仅仅是语法工具，更是情绪和节奏的载体。
   执行:
       *   **根据语境调整**: 拒绝教科书式的呆板标点，根据当前环境氛围和语境灵活运用。
       *   **紧张/愤怒/急迫**: 多用短句和**感叹号 (!)**。
           *   Example: “苏云！” (比“苏云。”更有张力)
       *   **压抑/思考/犹豫**: 多用**省略号 (...)** 或 **句号 (.)** 制造停顿和留白。
       *   **冷漠/陈述**: 使用简短的句号，切断情感连接。

## 4. 禁止项 (黑名单)
   禁止AI味描写:
       ❌ “像是有只手伸进胸腔里硬拽/抓碎” (生硬的肉体入侵描写)
       ❌ “像是被按下了暂停键” (陈旧的通感)
       ❌ “命运的齿轮开始转动” (陈词滥调)
       ❌ “无形的大手/铁钩” (老套且偷懒的比喻) -> 改为具体的物理质感（如：空气焊死、关节生锈、重力异常）。
   禁止老套比喻:
       ❌ “肾上腺素不要钱似的往外泵”、“肺像破风箱” (极度陈旧的网文套话) -> 改为直接描写痛感和心跳。
       ❌ “喉咙里发出风箱一样的喘气声” (AI常用烂梗) -> 改为“破碎的呜咽”、“浑浊的嘶吼”。
       ❌ “瞳孔猛缩”、“瞳孔收缩成针芒/针孔” (滥用的微表情) -> 改为直接描写动作僵硬、呼吸停滞或冷汗直冒。
   禁止否定式描写:
       ❌ “不再是风，是...” / “那不是...而是...” -> 直接写“风声骤变，夹杂着...”。
       ❌ “写下的不是字，是疯癫的诗” -> 改为直接描述“写下了癫狂的诗句”。
   禁止解释性指代:
       ❌ “...最后定格在墙角。那是满脸是血的赵三...” (禁止用“那是...”来指代刚看到的画面) -> 改为直接描写“...最后定格在墙角。满脸是血的赵三正...”。
   禁止抽象心理描写 (Concretization):
       ❌ “他强行进入了冷静状态/导演模式” (抽象概念) -> 改为生理+具体思维：“他咬破舌尖，用疼劲儿压住发软的膝盖。只要把杀人魔当成蹩脚演员……”
   禁止滥用词汇:
       ❌ “仿佛”、“似乎”、“宛如” (一章内出现超过3次即视为滥用)

## 5. 去AI化专项 (2026新增)
   禁止被动语态:
       ❌ “诊断书被揉成了一团” -> 改为主动：“他把诊断书揉成一团”。
   禁止“状语+地+动词”结构:
       ❌ “突兀地浮现”、“狠狠地摔下” -> 删去“地”，或改用动补结构：“直接浮现”、“摔得粉碎”。
   禁止特定烂梗/陈词:
       ❌ “嘴角扯出弧度/勾起一抹笑” -> 改为具体肌肉动作：“脸部肌肉抽动”、“硬挤出笑”。
       ❌ “撕心裂肺”、“歇斯底里” -> 改为生理描写：“咳得肺叶要炸开”、“喉咙里发出破风声”。
       ❌ “极致的”、“极其” -> 删去或换具体描写。
       ❌ “缩成虾米” -> 改为“整个人蜷起来”、“缩成一团”。
   禁止重复否定句式:
       ❌ “没有投影，没有屏幕” -> 直接写“字直接烧在视网膜上”。
       ❌ “不是……而是……” -> 直接写后者。

## 6. 风格迭代总结 (基于实战修改·2026.01)

### 6.1 电影感分镜 (Fast Cut)
   核心: 像电影剪辑一样使用短句。
   修正: 必须使用 3字及以上 的短语，严禁单双字。
   执行: 在高潮或转折处，放弃长句，连续使用短促有力的短语，制造呼吸感。
   Example:
       Before: 血液不是红色的，而是像墨汁一样深沉。
       After: 不是血......那是某种比夜色更深沉的墨汁。 (注：“不是血”为3字，合规)

### 6.2 动作颗粒度 (Micro-Actions)
   核心: 将一个笼统的动作拆解为2-3个具体的生理/物理细节。
   执行: 不要只写“他很兴奋”，要写“手抖”、“咬飞笔帽”、“肌肉抽搐”。
   Example:
       Before: 他激动地打开笔盖签字。
       After: 哆嗦着摸出来一支笔，牙齿咬飞笔帽。极致的兴奋让手抖得不成样子。

### 6.3 职业滤镜 (Professional Lens)
   核心: 主角的心理活动必须经过“导演/编剧”身份的过滤。
   执行: 面对死亡危机时，不写“恐惧”，写“机位完美”、“光线逆光”、“构图千载难逢”。
   Effect: 立住“疯批”人设，制造强烈的反差爽感。

### 6.4 设定软植入
   核心: 拒绝说明书式的设定抛出。
   执行: 让主角在使用能力的过程中，通过感官（痛感、消耗、视觉反馈）顺带引出设定。
   Example: “消耗了一滴墨水”比“系统提示：墨水-1”更自然。

## 7. 通俗化与受众适配 (2026.01新规)

### 7.1 低门槛原则 (Low Entry Barrier)
   目标受众: 低龄及大众读者。阅读过程应如流水般顺畅，无任何理解阻碍。
   词汇降维:
       ❌ 拒绝成语/书面语: “文质彬彬”、“裹挟”、“陈腐”、“触目惊心”。
       ✅ 拥抱白话: 改为“斯斯文文”、“带着”、“老旧/难闻”、“刺眼”。
   
### 7.2 生理/专业名词通俗化
   规则: 除非极其必要，否则将医学/解剖学名词替换为生活常识词汇。
   执行表:
       | 原词 (Bad) | 替换词 (Good) |
       | :--- | :--- |
       | 耳膜 | 耳朵 |
       | 脑浆/前额叶 | 脑子/脑袋 |
       | 肺叶 | 肺/胸口 |
       | 视网膜 | 眼睛 |
       | 声带 | 嗓子 |
       | 肾上腺素 | 心跳 |

### 7.3 “忽悠”专用词汇特权
   例外规则: 允许出现“符合空气动力学”、“次声波共振”、“磁场异常”等伪科学/专业术语的**唯一场景**：
   *   **场景**: 李严（或苏云模仿/嘲讽李严时）在编造借口、忽悠公众/媒体/普通人时。
   *   **目的**: 利用信息差制造“一本正经胡说八道”的黑色幽默效果。
   *   **限制**: 除此之外的旁白、心理活动及正常对话，必须严格遵守7.1和7.2的通俗化规则。
